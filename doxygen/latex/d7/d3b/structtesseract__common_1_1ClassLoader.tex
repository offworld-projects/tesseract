\doxysubsection{Detailed Description}
This is a wrapper around Boost DLL for loading plugins within Tesseract. 

The library\+\_\+name should not include the prefix \textquotesingle{}lib\textquotesingle{} or suffix \textquotesingle{}.so\textquotesingle{}. It will add the correct prefix and suffix based on the OS.

The plugin must be exported using the macro TESSERACT\+\_\+\+ADD\+\_\+\+PLUGIN. In the example below, the first parameter is the derived object and the second is the assinged symbol name which is used for looding Example\+: \mbox{\hyperlink{class__loader_8h_a97c9d4061ddaef2cc923033180af020f}{TESSERACT\+\_\+\+ADD\+\_\+\+PLUGIN(my\+\_\+namespace\+::\+My\+Plugin, plugin)}}
