Find tool center point provided in the manipulator info. 

If manipulator information tcp is defined as a string it does the following
\begin{DoxyItemize}
\item First check if manipulator info is empty, if so return identity
\item Next if not empty, it checks if the manipulator manager has tcp defined for the manipulator group
\item Next if not found, it looks up the tcp name in the \mbox{\hyperlink{structtesseract__environment_1_1EnvState}{Env\+State}} along with manipulator tip link to calculate tcp
\item Next if not found, it leverages the user defind callbacks to try an locate the tcp information.
\item Next throw an exception, because no tcp information was located.
\end{DoxyItemize}
